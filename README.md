# PaintRoom Demo – NCS Style

Local offline prototype built with Vite + React + TypeScript.

## Important Notice

- This app uses **synthetic demo colors** generated algorithmically.
- These are **not official NCS data**.
- No scraping is used.
- Intended for **investment prototype/demo purposes only**.

## Features

- Webcam auto start with image upload fallback
- Polygon wall mask creation/editing
- Synthetic NCS-style palette generator (~120 colors)
- Live wall recolor rendering with opacity and optional shading mode
- Save PNG output
- Save/Load mask JSON (normalized points)

## Install and run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
